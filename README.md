# Librar IST

* Android app for managing and crowd-sourcing free libraries. Uses [Android JetPack Compose](https://developer.android.com/jetpack) with Material 3.
* Created as uni project for the LPRO course.

## Used libraries
* Android Room
* Retrofit
* CameraX
* Dagger - Hilt
* Coil
* Bar Code Scanner
* Maps SDK
